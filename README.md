# Photon-Curtain Weak-Measurement Simulation

Code accompanying the manuscript "Weak-Measurement Reconstruction of the
Conditional Momentum Field for Atomic Matter Waves: A Dispersive
Photon-Curtain Protocol Extending Kocsis-Steinberg" by J. Fabianiak and
T. Deligiannis (2026).

Preprint: https://doi.org/10.5281/zenodo.19859212

## Contents

This repository contains the Python source used to perform the forward-modeled
reconstruction simulation and to generate Figures 2-5 of the manuscript.

## Requirements

Python 3.9 or later, with the following packages:

- numpy
- scipy
- matplotlib

Install with: `pip install numpy scipy matplotlib`
